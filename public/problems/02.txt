#TITLE 02 - Bakery Menu System
#SUMMARY Task 02: Managing Bakery Items using Classes and Constructors

## Problem Overview
Ali is running a bakery and he wants to create a set of bakery items to store in his menu file. Currently, he just wants to use a basic array (list) to store these items. Your task is to develop a script that manages the bakery menu using object-oriented principles, specifically classes and constructors.

---

### Core Requirements
To complete this task, your Python script must implement the following:

1.  **Class Definition**: Create a `BakeryItem` class with a constructor (`__init__` method) that initializes the `name`, `price`, and `category` of a bakery item. The category can be "Cakes", "Cookies", or "Sweets".
2.  **Print Functionality**: Create a basic function (or class method) that prints the details of the bakery item.
3.  **Data Structure**: Initialize an empty list named `menu`.
4.  **Outcome Verification**: 
    *   Create 3 sample bakery items using the class constructor.
    *   Add each of the 3 items to the `menu` list.
    *   Use your print function to display the details of the items.

---

### Expected Output Structure
When executed, your program should display the details of the bakery items clearly, for example:
```text
Item: Chocolate Cake | Category: Cakes | Price: $15.00
Item: Chocolate Chip Cookie | Category: Cookies | Price: $2.50
Item: Vanilla Fudge | Category: Sweets | Price: $3.00
```

---

### Implementation Guide
> [!TIP]
> **Class Constructors**: Use the `__init__` method in Python to set up the initial attributes (`name`, `price`, `category`) of your object when it is instantiated.

> [!IMPORTANT]
> A single Python script should be used to complete this task.
